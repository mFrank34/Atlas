# Atlas

A comprehensive character creation and editing tool for Dungeons & Dragons that helps players build and 
manage their character sheets digitally. Atlas simplifies tracking stats, abilities, and progression, 
making character management faster and more intuitive.

## Features

- **Character Creation**: Build D&D characters from scratch with an intuitive interface
- **Character Management**: Edit and update existing characters as they progress
- **Stats Tracking**: Monitor ability scores, hit points, and other vital statistics
- **Ability Management**: Keep track of skills, spells, and special abilities
- **Progression Tracking**: Record character level ups and advancement
- **Digital Character Sheets**: Store and access your characters digitally

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Apache Maven 3.6+

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mFrank34/Atlas.git
cd Atlas
```

2. Build the project using Maven:
```bash
mvn clean install
```

3. Run the application:
```bash
mvn exec:java
```
